# Dynamic Form Builder

A Laravel-based web application that allows administrators to create dynamic forms and enables employees to submit responses through a simple and intuitive interface.

---

## Features

### Admin

- Authentication (Login / Register)
- Dashboard overview
- Create, update and delete forms
- Activate / deactivate forms
- Manage dynamic form fields
- Configure field options
- View employee submissions

### Employee

- Register and login
- Browse active forms
- Submit responses
- View submitted forms

## Installation

Clone the repository

```bash
git clone <repository-url>
cd dynamic-form-builder
```

Install PHP dependencies

```bash
composer install
```

Install Node dependencies

```bash
npm install
```

Create environment file

```bash
cp .env.example .env
```

Generate application key

```bash
php artisan key:generate
```

Configure your PostgreSQL database inside `.env`

```env
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=dynamic_form_builder
DB_USERNAME=postgres
DB_PASSWORD=your_password
```

Run migrations and seed demo data

```bash
php artisan migrate:fresh --seed
```

Build frontend assets

```bash
npm run build
```

Start development server

```bash
php artisan serve
```

Visit

```
http://127.0.0.1:8000
```

## Architecture

The project follows a layered architecture.

```
Request
      │
      ▼
Controller
      │
      ▼
Service
      │
      ▼
Model
      │
      ▼
Database
```

Business logic is separated into dedicated service classes while request validation is handled by Laravel Form Requests.
